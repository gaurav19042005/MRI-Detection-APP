# 🧠 MRI Brain Tumor Detection System

This project is a Deep Learning-based system that detects and classifies brain tumors from MRI images. It also stores prediction history and generates downloadable reports.

---

## 🚀 Features

- 🧠 MRI Tumor Classification using CNN (TensorFlow/Keras)
- 📊 Confidence Score & Probability Visualization
- 🗃️ SQLite Database for storing prediction history
- 📄 Downloadable MRI Report (Text format)
- 🏥 Hospital-style Streamlit Dashboard

---

## 🧪 Tumor Classes

- Glioma
- Meningioma
- Pituitary
- No Tumor

---

## ⚙️ Tech Stack

- Python
- TensorFlow / Keras
- Streamlit
- SQLite3
- Pandas & NumPy
- PIL (Image Processing)
